# Launchers

## Description

Made Rest API endpoints to be used in a front-end project.

## Installation

```bash
$ git clone git@github.com:amirpoint/launchers.git --depth 1
$ mv launchers my-project
$ cd my-project/server
$ npm i
$ cd my-project/client
$ npm i
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm watch

# unit test
$ npm test

# test coverage
$ npm test-watch
```
